========================
        ОПИСАНИЕ        
========================

Данный скрипт предназначен для определения местоположения (города) пользователей по их IPv4 адресам. Приложение на вход получает .csv файл, после чего генерирует три новых .csv файла, которые содержат: уникальные IPv4 адреса из входного файла, результат работы программы, ошибки при определении местонахождения.

========================
    КАК ИСПОЛЬЗОВАТЬ
========================
— Поместите исходный .csv файл в директорию файла main.py
— Запустите main.py
— Получите результат.

Важно! Для корректной работы скрипта необходимо стабильное подключение к сети интернет.

========================
       НАСТРОЙКА
========================
Настройка скрипта осуществляется через файл config.ini.
Файл конфигурации состоит из трех разделов: FILES, COLUMNS, META.
 — Первый раздел содержит четыре опции:
	- input_file (по умолчанию имеет значение data.csv) — имя входного файла.
	- interim_file (по умолчанию имеет значение interim.csv) — имя промежуточного файла, в который будут записаны уникальные IPv4 адреса.
	- output_file (по умолчанию имеет значение data.csv) — имя выходного файла, в который будут записаны результаты работы скрипта.
	- error_file (по умолчанию имеет значение error.csv) — имя файла, куда будут записаны невалидные/нераспознанные IP.
 — Второй раздел содержит две опции:
	- ip_column (по умолчанию имеет значение customer_ipaddress) — имя столбца с IP адресами во входном и выходном файле.
	- city_column (по умолчанию имеет значение city) — имя столбца с городами в выходном файле.
 — Третий раздел содержит одну опцию:
	- step (по умолчанию имеет значение 50) — количество IP адресов, обрабатываемых за один "пробег".
